Opens a web browser to fill out the Amphitheatre survey

## Usage
```
amp survey [options]
```

## Options

```
      --id='hats': Survey ID for survey command to open.
```

> Use "amp options" for a list of global command-line options (applies to all commands).

## Environment vars

* `AMP_ID` (same as `--id`)
